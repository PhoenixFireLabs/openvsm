1 OpenVSM
2 =========
3 
4 OpenVSM is a model sumulation engine with C and Lua API for Proteus 7/8 CAD
5 
6 Written in C and gcc-ready (mingw/cygwin) for Linux and Windows
7 
8  - You can create your model as standalone DLL or use DLL and Lua script together
9  - Function prototypes have the same syntax in both API
10  - Designed with hope to make simulation as simple as possible for electronics enthusiasts
11 
12 
13 Visit 'example' directory for sample project files
14 
15 Version
16 ----
17 0.1
18 
19 How to
20 --------------
21 
22  - Install mingw32 for you platform
23  - Clone https://github.com/Pugnator/openvsm.git
24  - Change to the cloned directory
25  - Issue "make" command in Linux or "mingw32-make" under Windows
26  - Copy lua53/lua.dll to %windir%\system32 directory
27  - Create environment variable containing path to the script directory,
28 issuing the following command:
29 
30 ```bat
31 setx LUAVSM "C:\script\"
32 ```
33  - In this case you should place your model script to c:\script directory,
34  note the trailing backslash
35  - Your script should be named after the name of model DLL, for example:
36 
37 ```model.dll
38 model.dll.lua```
